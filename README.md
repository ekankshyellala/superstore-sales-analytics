# Superstore Sales Analytics 📊

## Overview
Analyzed 500K+ Superstore sales records across US & Canada.
Built interactive Power BI dashboard tracking sales, profit & operational metrics.

## Dataset
- Records: 500K+ transactions
- Countries: United States, Canada
- Regions: West, East, Central, South
- Categories: Furniture, Office Supplies, Technology

## Tools Used
- Python - Data cleaning, EDA (Pandas, NumPy)
- SQL - 23 advanced queries with window functions
- Excel - Pivot tables & analysis
- Power BI - Interactive dashboards

## SQL Queries (23 total)
✅ Basic aggregations (Profit/Sales by Customer, Region, Category)
✅ Window functions (DENSE_RANK with PARTITION BY)
✅ Date analysis (Monthly trends)
✅ Discount impact on profitability
✅ Shipping mode efficiency
✅ Product & Sub-category rankings
✅ Regional & city-level analysis

## Key Insights
- Discount inversely impacts profit - higher discounts = lower margins
- Monthly seasonality visible - peaks in certain months
- West region most profitable
- Technology category drives highest profit
- Standard shipping most popular but lower margin
- Geographic concentration in US East Coast
